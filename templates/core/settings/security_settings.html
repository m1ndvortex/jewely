Security Information

Multi-Factor Authentication

MFA adds an extra layer of security by requiring a second form of authentication in addition to your password. We support TOTP (Time-based One-Time Password) apps like Google Authenticator, Authy, or 1Password.

Password Security

All passwords are encrypted using industry-standard Argon2 hashing. We enforce strong password requirements including minimum length, complexity, and common password checks.

Data Encryption

All sensitive data including API keys and credentials are encrypted at rest using AES-256 encryption. All communications use TLS 1.3 for encryption in transit.

Security Best Practices

  • • Enable MFA for all manager and owner accounts
  • • Use strong, unique passwords for all accounts
  • • Regularly review user access and remove unused accounts
  • • Keep your browser and devices updated with security patches
  • • Never share login credentials or leave devices unlocked
  • • Report any suspicious activity immediately
